MultiApp merupakan sebuah project kuliah yang menerapkan Session, LBS, dan masih banyak fitur lainnya. Di aplikasi ini, fitur yang tersedia antara lain :
- Login
- Stopwatch
- LBS Tracker
- Time Conversion
- Site Recommendations
- List Site Favorite
- List Developer Team
- About This App
